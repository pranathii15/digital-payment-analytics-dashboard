# digital-payment-analytics-dashboard
A data analytics platform for analyzing digital payment transactions using SQL, Python, Pandas, NumPy, and Power BI to generate spending insights, fraud-risk indicators, and interactive dashboards.

<img width="1536" height="1024" alt="image" src="https://github.com/user-attachments/assets/e72ccd87-5b3d-4228-9234-bbc1fd559a5d" />


## Features

- ETL Pipeline using Python
- Data Cleaning and Transformation
- Fraud Detection Analysis
- SQL Schema Design
- Power BI Dashboards

## Dashboards

### Dashboard 1: Digital Payment Analytics
- Total Transactions
- Total Amount
- Fraud Transactions
- Monthly Transaction Volume
- Fraud by Merchant Category
- Fraud by State
- Transaction Amount by Sender Bank

### Dashboard 2: Fraud Analytics
- Fraud Rate
- High Value Transactions
- Fraud by Device Type
- Fraud by Network Type
- Fraud by Age Group
- Fraud by State

### Dashboard 3: Customer & Transaction Behavior
- Average Transaction Amount
- Weekend Transactions
- Age Group Analysis
- Device Type Analysis
- Network Type Analysis
- Day-wise Transaction Analysis
